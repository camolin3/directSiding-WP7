directSiding-WP7
================

Repositorio oficial de la versión de directSIDING para Windows Phone 7 y 8.

Si utilizas WP7 tendrás problemas por el nuevo certificado HTTPS que están utilizando.
Se requiere que sigas los siguientes pasos:

Desde tu computador
-------------------
1. Descarga [el certificado .pem](/GlobalSignOrganizationValidationCA-G2.pem) a tu Escritorio
2. Envíate un email con el certificado como archivo adjunto (al email que revisas en tu teléfono)

Desde tu celular
----------------
3. Revisa el email que te enviaste desde la aplicación de emails.
4. Descarga el archivo adjunto. Aparecerá un ícono de un escudo.
5. Abre el archivo adjunto y verás un mensaje de que el certificado se ha instalado exitosamente.

Y listo, ya puedes usar directSIDING desde WP7.
